# 核心特性

MSM 提供了一套完整的服务管理解决方案，以下是详细的功能特性介绍。

## 🚀 服务管理

### 一键安装

- 支持从 GitHub Releases 自动下载最新版本
- 支持自定义下载源（镜像加速）
- 自动校验文件完整性（SHA256）
- 智能解压和部署

### 灵活启停

- 一键启动/停止/重启服务
- 支持自定义启动参数
- 自动检测端口冲突
- 进程守护和自动重启

### 多内核管理

- 支持安装多个内核版本
- 一键切换不同版本
- 保留历史版本便于回退
- 版本对比和更新提示

### 实时状态监控

- 服务运行状态（运行/停止/错误）
- 进程 PID 和运行时间
- CPU 和内存使用率
- 端口占用情况

## 📝 配置管理

### 在线编辑器

- 基于 Monaco Editor（VS Code 同款）
- 语法高亮（YAML/JSON）
- 代码自动补全
- 错误提示和警告
- 代码格式化

### 配置验证

- 保存前自动验证
- 实时错误提示
- 语法检查
- 逻辑校验

### 配置模板

- 内置常用配置模板
- 一键应用模板
- 自定义模板保存
- 模板分类管理

### 配置对比

- 可视化 Diff 视图
- 高亮显示差异
- 支持任意两个版本对比
- 支持合并操作

## 🔄 历史管理

### 自动快照

- 每次保存自动创建快照
- 记录操作人和时间
- 保存配置内容和元数据
- 支持添加备注说明

### 版本回滚

- 一键回滚到任意历史版本
- 回滚前预览差异
- 回滚后自动重启服务
- 失败自动恢复

### 稳定版本标记

- 标记测试通过的稳定版本
- 快速回滚到最近稳定版
- 版本标签和分类
- 版本搜索和筛选

### 智能清理

- 自动清理过期快照
- 保留重要版本
- 可配置保留策略
- 手动删除支持

## 📊 监控与日志

### 实时日志流

- WebSocket 实时推送
- 自动滚动显示
- 支持暂停和恢复
- 日志颜色高亮

### 日志搜索

- 关键词搜索
- 正则表达式支持
- 时间范围筛选
- 日志级别过滤

### 日志导出

- 下载日志文件
- 支持时间范围导出
- 多种格式支持
- 压缩打包下载

### 资源监控

- CPU 使用率曲线
- 内存使用趋势
- 网络流量统计
- 磁盘空间监控

## 👥 用户管理

### 多用户支持

- 创建多个用户账号
- 用户信息管理
- 头像和个人资料
- 密码修改和重置

### 角色权限

四级权限体系：

1. **Admin（管理员）**
   - 所有操作权限
   - 用户管理
   - 系统设置

2. **Operator（运维人员）**
   - 服务启停
   - 配置编辑
   - 日志查看

3. **Viewer（只读用户）**
   - 状态查看
   - 日志查看
   - 无修改权限

4. **Guest（访客）**
   - 仅查看概览
   - 最小权限

### 操作审计

- 记录所有操作
- 操作人和时间
- 操作详情和结果
- 审计日志导出

### 安全控制

- 登录失败锁定
- 密码强度要求
- 会话超时控制
- IP 白名单（可选）

## 🔐 安全特性

### 认证授权

- JWT 令牌认证
- Token 自动刷新
- 双令牌机制（Access + Refresh）
- 令牌黑名单

### 数据加密

- 密码 bcrypt 加密
- HTTPS 传输加密
- WebSocket 加密（WSS）
- 敏感信息脱敏

### 访问控制

- 基于角色的权限控制（RBAC）
- 细粒度权限管理
- API 级别权限校验
- 前端路由守卫

### 安全审计

- 登录日志
- 操作日志
- 错误日志
- 安全事件告警

## 📦 部署方式

### 单二进制部署

- 前后端打包成一个文件
- 无需额外依赖
- 开箱即用
- 体积小（约 15-23MB）

### Docker 部署

- 官方 Docker 镜像
- Docker Compose 支持
- 多架构支持（amd64/arm64）
- 自动更新

### 传统部署

- 前后端分离部署
- Nginx 反向代理
- Systemd 服务管理
- 自动启动配置
